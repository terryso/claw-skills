# Claw Skills

Collection of AI agent skills for Claude Code, OpenClaw, and other skill-compatible agents.

## Skills

### peekaboo-cli

macOS UI automation CLI tool for screen capture, window control, element clicking, text input, and more.

```bash
# Install
cp -r peekaboo-cli ~/.claude/skills/
```

### polyv-live-cli

保利威直播服务管理工具，支持频道管理、推流操作、商品管理、优惠券等功能。

```bash
# Install
cp -r polyv-live-cli ~/.claude/skills/
```

## Installation

Copy the skill folder to your agent's skills directory:

```bash
# Claude Code
cp -r <skill-name> ~/.claude/skills/

# OpenClaw
cp -r <skill-name> ~/.openclaw/skills/
```
